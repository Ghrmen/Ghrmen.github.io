语雀文本编辑器的功能测试

Ghrmen2025-07-132025-07-14

此前使用 Hexo 去建立好了我的个人博客，经过初步的使用呢，Hexo 确实有很大的功能延展性和可塑性，但是相较于其他的博客部署，部署不了在线编辑器这一点无疑是 Hexo 的巨大缺陷，这意味着使用 Hexo 的博客作者们只能通过其他编辑器来编写日志，在一众编辑器中我选择使用在线编辑器语雀，基于其强大的文本编辑能力，多样的文本插件功能和方便的文档类型转化能力，基本可以满足程序博客的需求。正好以本篇文章测试语雀的各项实用功能，同时将其挂载到博客上查看有多少功能能被相应的挂载上去

篇幅有限就不继续列举剩下的内容了，可以看到语雀搭载的模块功能非常多样化且强大，确实是一款优秀的在线编辑器，最重要的是免费😆

对语雀有兴趣的可以移步语雀网站实际应用一下https://www.yuque.com/dashboard

最后呢 hexo 不能挂载在线编辑器，那么我现在写博客就需要通过语雀来编写日志，然后转移文件到自己电脑的博客路径内，再通过命令框进行挂载，这么复杂的行径对于一个程序员来说实在是太不优雅了，我尝试过使用 github 中的 yuque-hexo 插件实现博客路径文件同步语雀知识库，并通过脚本定期自动挂载，但是在云服务器的租借价格和编辑器的会员价格面前败下阵来，穷鬼是这样的…..虽然没法使用 yuque-hexo ，但是编写脚本实现手动导出自动同步还是很有机会的，后续会使用 Python 写个脚本尝试一下。
